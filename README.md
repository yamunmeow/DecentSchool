# DecentralSchool 🏳️‍🌈

A colorful, educational website built with Astro for decentralized learning content.

## Features

- 🌈 Rainbow-themed design with animations
- 📚 Educational texts and articles
- 😂 Coding memes gallery
- 🎨 Creative illustrations showcase
- 💻 Code Lab for student contributions
- 📰 Education news and updates
- 📱 Community blog

## Projects Structure

DecentralSchool/
├── public/              # Static assets
├── src/
│   ├── components/      # Reusable components
│   ├── layouts/         # Page layouts
│   ├── pages/          # Page routes
│   │   ├── index.astro
│   │   ├── texts.astro
│   │   ├── memes.astro
│   │   ├── illustrations.astro
│   │   ├── code-lab.astro
│   │   ├── news.astro
│   │   └── blog.astro
│   └── styles/         # Global styles
├── astro.config.mjs    # Astro configuration
└── package.json        # Dependencies

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
